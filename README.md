### 配置项说明:

  

| 配置项            | 说明                 | 是否能为空 |
| ----------------- | -------------------- | ---------- |
| canal.server      | canal server端ip地址 | 非空       |
| canal.port        | canal server端端口   | 非空       |
| canal.destination | canal server端实例   | 非空       |
| canal.enabled     | 是否启用canal        | 非空       |
